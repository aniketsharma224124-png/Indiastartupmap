## Current structure:

### `Home.jsx`
1. **Left Column (`lg:grid-cols-[55%_43%]`)**
   - Contains:
     - `IndiaMap` (startup mode) or `InvestorMapSVG` (investor mode)
     - `Buy More Intro Credits` section (rendered right below the map in the same column, ONLY in 'startup' mode)
2. **Right Column (`lg:grid-cols-[55%_43%]`)**
   - Contains:
     - `StatePanel` (if state selected in investor mode)
     - `StartupListingPanel` (in startup mode, contains "List Your Startup" and "Browse Startups" tabs)
     - Empty state prompt (in investor mode)

### Requirements:
1. **"add more space between map section and buy credits so that listing form and buy credits will be end on same level"**
   - The user wants the bottom of the "buy credits" section to align with the bottom of the "listing form" (which is in the `StartupListingPanel` in the right column).
   - Currently, they are in the left column. We can use flexbox space-between or margin-top on the credits section, or use `flex flex-col justify-between h-full` on the left column container, assuming the right column drives the height. Wait, `items-start` is used in the grid, so the columns don't stretch to equal height. We can change `items-start` to `items-stretch` to make both columns equal height, then make the left column a flex container `flex flex-col justify-between`, so the map is at the top and the buy credits section is pushed to the bottom.
